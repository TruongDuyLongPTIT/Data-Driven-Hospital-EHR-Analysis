  • u tin, cn bit mi spark worker c bao nhiu core. Gi s, mi worker c 2 cores th tng s core s l 6.
  • 10 partition s c chia thnh 10 task.

=> Spark cluster s phn phi nh sau:

  • Lp lch thc thi 10 task ny: 6 tasks u phn cho 6 cores thc thi trc -> sau , ci no xong trc th s c phn task cho thc hin tip n khi ht task th thi.

  • Nu bng qu nh khng cn chia task.

    • S lng ph thi gian v nng lc tnh ton. V d: C 6 cores nhng m ch c 2 partitions, th tc l c 2 cores lm vic thi cn 4 cores s ngi chi (nh vy s khng tn dng c 4 thng ny lm gim thi gian)

7. Cn fetch_size th sao, n l g v da vo u nh gi tr cho n?

  • Postgres khng th c data t disk ri chuyn cho Spark lun c, m n cn phi load ln RAM trc, xong mi chuyn sang cho Spark.

  • V sao li phi load ln RAM trc? L do l, n cn phi p dng filter, join, index scan,... kiu nh l phi lc d liu theo iu kin where ln RAM trc, ri d liu mi stream sang cho Spark c.

  • Vn xy l y l: i khi partition kh to, to hn c RAM mc RAM c th x l tt, do , cn phi chia thnh n v nh hn l cc FETCH load t DISK ln RAM ri stream sang cho Spark thng qua cc connection.

! Lu : Fetch_size khng tnh bng byte m tnh bng s record, v du: fetch_size = 10000. Tc l mi fetch s gm 10000 dng (records)

  • Spark worker khng ch n khi gom fetch thnh 1 partition ri mi x l, m mi Fetch n nhn c th n x l ngay lp tc.
  • Th nn, v bn cht, l cc Spark Worker Core x l tng fetch, ch khng phi x l c partition cng lc.
  • Partition l gii hn v mt logic thi, n gip m bo:
    • 2 cores khc nhau khng c load cc fetch ca cng 1 Partition.
    • Mi cores x l 1 Partition ri lu thnh .parquet th .parquet mi gi c ci khong [min, max] tng ng vi khong [min, max] ca partition ban u => T , mi tm kim c nhanh hn.
    • C nh c size mi Partition, Partition khai bo gm nhng bn ghi no th chc chn ch ly nhng bn ghi . Cn fetch th kch thc mi fetch hn xui, tuy l c khai bo fetch_size = 10000, nhng c khi fetch ch gm 9800 dng, n ty thuc vo RAM kh dng ca Postgres Server. Tm li, fetch_size ch l s dng ti a ca 1 fetch, cn kch thc th mi fetch li khc nhau.

Hnh nh v d:

  • 1 Table chia lm 3 Partitions
  • Mi Partition chia lm nhiu Fetch
  • Cc Fetch ca cc Partition khc nhau c load ln RAM, ri Spark Master iu phi cc Fetch ny ti cc Spark Worker x l (qu trnh ny l song song - tc l 3 fetch c load ln RAM cng lc v cng c x l cng lc nh trong hnh)
